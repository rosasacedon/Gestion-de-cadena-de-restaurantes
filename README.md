En el equipo 1 podemos encontrar a Sergio Cañadillas y Víctor Martín; Equipo 2 formado por Ana Díaz y Guiomar Pareja; y, finalmente, el equipo de trabajo 3 por Marta Gallego y Rosa María Sacedón.
